GRAVITY FLIP — FULL PROJECT OVERVIEW

---

1. CORE CONCEPT

Gravity Flip is a fast-paced 2D endless runner built around a single mechanic:
tap to flip gravity.

The player controls a constantly moving character inside a horizontal corridor filled with hazards. By flipping gravity, the player alternates between running on the floor and the ceiling to avoid obstacles.

The core experience is:

- Immediate start
- High-speed gameplay
- Frequent failure
- Instant retry
- Skill-based progression

The game is designed for:

- Short sessions (15–60 seconds)
- High replayability
- Addictive loop behavior

---

2. CORE GAME LOGIC

Player Movement System

- Character auto-runs forward at constant speed
- No horizontal control
- Vertical state toggles between:
  - Floor (gravity down)
  - Ceiling (gravity up)

Gravity Flip Mechanic

- Tap input triggers gravity inversion
- Flip animation duration: ~0.15–0.25 seconds
- Player transitions smoothly between surfaces

Collision System

- Collision types:
  
  - Instant death obstacles
  - Power-up triggers
  - Collectible pickups

- Hit detection:
  
  - Bounding box collision (simple rectangle or capsule)

Game Speed System

- Base speed increases over time
- Speed curve:
  - Linear increase first 30 seconds
  - Exponential increase after 60 seconds

---

3. MAP / LEVEL STRUCTURE

Endless Procedural Corridor

The game uses a procedural generation system:

- Predefined obstacle chunks
- Randomized sequence
- Difficulty-based spawning

Corridor Layout

- Fixed horizontal scrolling
- Two surfaces:
  - Floor
  - Ceiling

Chunk System

Each chunk:

- Length: 1–3 seconds of gameplay
- Contains obstacle pattern
- Tagged by difficulty level
